## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues. Use the gh CLI for all operations.
External PRs are not a triage surface.
See docs/agents/issue-tracker.md.

### Triage labels

The five canonical triage roles use their default names (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix).
See docs/agents/triage-labels.md.

### Domain docs

Single-context — one CONTEXT.md at the repo root + docs/adr/ for architectural decisions.
See docs/agents/domain.md.
