# 酷残未来研究院（CFI）— Context

## Project

酷残未来研究院（Cripping Future Institute）官方网站。一个由残障人发起并领导的独立民间残障研究机构的内容管理系统网站。

## Domain Language

| Term | Definition |
|------|-----------|
| 酷残未来研究院 | Cripping Future Institute（CFI） |
| 蒹葭计划 | 鼓励残障者做基于自身需求和权利保障的研究 |
| 经纬计划 | 鼓励青年研究者与社群伙伴一起做研究，更加理论与结构化 |
| 拾遗计划 | 整理盲文成数字化、口述史 |
| 酷残学院 | CFI Academy，独立课程平台 |

## Tech Stack

- Python 3.10+ stdlib only
- SQLite
- Zero third-party deps
- Linux + Nginx + systemd deployment
